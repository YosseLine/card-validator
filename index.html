ValidateJS

Una librería que te ayudará a validar tarjetas de crédito
Ver DEMO


¿Por qué usar ValidateJS?

Esta librería te permitirá validar tarjetas de crédito junto con datos como: Nombre completo, código de validación y fecha de expiración de tarjeta. Todo lo hará de una forma fácil.

Extensibilidad

ValidateJS es una herramienta completamente extensible y funciona bien con otras librerías. Continúa leyendo para saber cómo aplicar ValidateJS a tu proyecto.

Cómo incluir ValidateJS a tu proyecto

1- Como primer requisito, tu archivo html deberá contar con un formulario que cumpla con las siguietes característivas:

* Tener inputs con un atributo llamado 'name'.
* Cada atributo name deberá tener un valor igual al del ejemplo de la imagen.
* tu index deberá tener un botón, con el cuál se hará el llamado a la librería.

2- El segundo requisito, claro está, sería llamar a la librería destu tu archivo js el cuál deberá estar vinculado a tu archivo html.

¡Y listo! Solo sigue estos sencillos pasos, deja volar tu imaginación y construye grandes cosas!

Con pocos pasos, podrás validar una tarjeta.
